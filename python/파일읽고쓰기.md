# 파일 읽고쓰기

## 파일 읽기

### 새로운 파일 생성

```python
f = open('file_name.txt', 'r')	# 읽기모드(default)
f = open('file_name.txt', 'w')	# 쓰기모드(새로 생성)
f = open('file_name.txt', 's')	# 추가모드(기존 파일에 추가)
```

### 새로운 파일 생성(파이썬 스타일)

```R
with open('file_name.txt', 'w') as file:
	<code>
```

### 내용 추가(\<code> 부분)

```python
file.write('hello')

```
