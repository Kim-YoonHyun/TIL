- git stash

  모든 커밋 시점을 복원이 가능. 되돌리기 가능.
  그래서 일반적으로 모든 작업을 할 때는 커밋하면서 이어나감.
  가끔 작업을 하던 도중 어떠한 것을 반영해야할 때가 있음.
  작업 하던 것을 잠시 보관 > stash

  ```bash
  $ git stash
  ```

  임시공간 보관하고

  ```bash
  $ git stash pop
  ```
