## tmux

### 2.1.1. 디렉토리 생성

```bash
mkdir aa
```

aa라는 이름의 디렉토리 폴더 생성.

### 2.1.2. -p 

중간 디렉토리 생성.

```python
mkdir -p a/aa
```

a 라는 중간 경로용 폴더를 자동으로 생성 후 aa 라는 이름의 디렉토리 폴더 생성.
-p 옵션이 없는 경우 중간 경로 폴더인 a 가 없어서 에러 발생.

### 2.1.3. ~

```bash
mkdir ~ aa
```
