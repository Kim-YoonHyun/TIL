## 2.1. 줄

### 2.1.1. 번호 붙이기

```bash
:set nu
```

### 2.1.2. 특정 번호 라인 이동

```bash
# :set nu 를 미리 설정해야함
:86
```

### 2.1.3. 줄 삭제

```bash
dd
```

### 2.1.4. 여러 줄 동시 삭제

```bash
지우려는 첫 번째 줄에서 v
지우려는 마지막 줄에서 d
```
